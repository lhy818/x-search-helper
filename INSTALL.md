# X Search Helper - 安装和使用指南

## 🚀 快速开始

### 方法一：开发模式安装（推荐）
1. **下载项目**：获取整个 `x-search-helper` 文件夹
2. **打开 Chrome**：进入 `chrome://extensions/`
3. **开启开发者模式**：点击右上角开关
4. **加载扩展**：点击"加载已解压的扩展程序"
5. **选择文件夹**：选择 `x-search-helper` 文件夹

### 方法二：使用 ZIP 文件
1. **下载 ZIP**：`x-search-helper.zip`
2. **解压文件**：解压到任意目录
3. **按方法一安装**：加载解压后的文件夹

## 🎯 功能演示

### 基本使用流程
```
1. 打开 X/Twitter 网站 (x.com 或 twitter.com)
2. 点击扩展图标 🔍
3. 选择模板分类 → 点击"使用"
4. 自动填充搜索框并搜索
5. 在结果页面选择"Latest"筛选
```

### 模板分类
- **中文圈**：5个模板，找中文爆款内容
- **日区**：5个模板，监控日本热门话题  
- **全球/美区**：5个模板，追踪国际趋势

### 高级功能
- **自定义关键词**：在模板基础上添加个性化关键词
- **自定义模板**：保存自己的搜索语法
- **模板管理**：编辑、删除自定义模板

### 技术要求
- **Chrome 88+**（支持 Manifest V3）
- **现代浏览器**（基于 Chromium 的浏览器都支持）
- **无需特殊权限**：仅需 storage 和 activeTab 权限

## 🐛 故障排除

### 常见问题

#### 1. 扩展图标不显示
- 检查是否成功加载扩展
- 重启 Chrome 浏览器
- 重新加载扩展

#### 2. 搜索框无法识别
- 确保在 X/Twitter 网站
- 刷新页面后重试
- 检查控制台错误（F12）

#### 3. 模板无法应用
- 检查网络连接
- 确认 X/Twitter 网站正常访问
- 尝试其他模板

#### 4. 自定义模板不保存
- 检查 Chrome 存储权限
- 清除扩展数据后重试

### 调试方法
1. **检查扩展状态**：`chrome://extensions/`
2. **查看控制台**：在 X 页面按 F12
3. **检查弹出窗口**：右键扩展图标 → "检查弹出内容"

## 📱 使用技巧

### 最佳实践
1. **结合使用**：先用模板搜索，再添加关键词细化
2. **定时检查**：每小时检查一次最新内容
3. **分类关注**：按兴趣关注不同分类模板
4. **保存常用**：将常用模板保存为自定义模板

### 搜索优化
- **时间筛选**：模板已包含时间范围，无需额外设置
- **语言过滤**：模板已设置语言，确保内容相关性
- **质量筛选**：通过点赞数过滤低质量内容

## 📚 模板语法参考

### X/Twitter 搜索运算符
- `lang:zh-cn` - 中文内容
- `min_faves:500` - 最少500赞
- `within_time:4h` - 4小时内
- `-is:retweet` - 排除转推
- `filter:images` - 只显示图片
- `filter:native_video` - 只显示视频

### 组合使用示例
```
# 中文AI内容，500+赞，4小时内
"AI" OR "大模型" lang:zh-cn min_faves:500 within_time:4h

# 日本图片内容，2000+赞，4小时内  
lang:ja min_faves:2000 filter:images within_time:4h

# 英文技术内容，10000+赞，12小时内，排除链接
lang:en min_faves:10000 -filter:links within_time:12h
```
